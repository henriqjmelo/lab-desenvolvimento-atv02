# Histórias de Usuários Sistema de Aluguel de Carros

Eu como cliente quero alugar um carro pela internet. Para isso realizo meu cadastro no sistema, informando meus dados pessoais, e faço login para acessar a plataforma.
Quero acompanhar o status do pedido, além de modifica-lo ou cancela-lo antes da aprovação.
